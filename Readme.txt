The main folder contains the python files. It needs a data file in the DataFiles folder to run. Selected results can be found in the Results folder. A copy of the project report is also in the main folder.

To run the program use "python project1.py".

project1.py: The main program

printer_functions.py: prints the figure 211 to file

support_functions: creates the data structures(files and folders). Also sets up the dataset using the Franke function. The dataset from real world data is set up in a method in the main project1.py program.

analysis_functions: functions used for statistical analysis
